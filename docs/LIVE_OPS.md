# Live Ops

## Live model

- Frontend-only deployed game with persistent leaderboard and challenge sharing
- No heavy live-service economy at present
- Daily challenge is the closest current live cadence system

## Operational levers

- Daily seed logic
- Leaderboard integrity
- Challenge-link virality
- Starter loadout and perk balance

## Safeguards

- Do not ship balance changes that invalidate leaderboard trust casually
- Preserve seed determinism where modes promise it
- Keep live issues triaged against player trust first, novelty second
