# Quality Bar

## Experience standard

- The game should feel instantly readable on desktop and mobile
- Comedy should enhance play, not obscure clarity
- Runs should feel punchy, chaotic, and fair enough to learn from

## Craft standard

- Mechanics must be legible before they are clever
- New content should add differentiation, not noise
- Social features should feel playful and polished

## Release bar

- `npm run build` passes
- Core run loop remains stable
- UI copy still sounds like Call of Doodie
- No major regression to pacing, controls, or leaderboard integrity
